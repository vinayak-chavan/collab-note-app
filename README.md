
# Collab Note App

## Tech Stack

**Client:** React, 

**Server:** Node, Express, Socket.io

## Installation

Install project with npm

## Live URL

https://65bdfb7af55590687e48cd59--aesthetic-moonbeam-e72f2f.netlify.app/

For frontend
```bash
  cd client
  npm install
  npm run start
```

For backend
```bash
  cd server
  npm install
  npm run start
```
